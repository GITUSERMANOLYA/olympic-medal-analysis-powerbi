# Olympic Medals – Top 10 Nations (Power BI)

## Overview
This project explores the medal composition of the top 10 Olympic nations using historical Olympic data (1896–2022).

## Key Insight
- The United States leads by a significant margin in total medals.
- European nations show a more balanced gold–silver–bronze distribution.
- Medal dominance varies strongly by nation rather than event volume alone.

## Tools Used
- Power BI
- CSV data (Olympic Games dataset)

## Dataset
Source: Kaggle – Olympic Games Medals (1896–2022)
